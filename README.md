# 🦖 JS Dino Game

A browser-based clone of the classic Chrome Dino game, written in **HTML5 Canvas + JavaScript**.  
Includes a **bot mode**, sound effects, and a help/options menu.

---

## 🎮 Features

- **Classic endless runner gameplay**
- **Keyboard & mouse controls**  
  - Jump: `SPACE` / `↑ Arrow` / Mouse Tap  
  - Crouch: `↓ Arrow`
- **Help / Options menu**
  - Toggle **Bot Mode** (Autoplay)
- **Idle Demo Mode** – starts automatically after inactivity
- **Dynamic ground, clouds, cacti, pterodactyls, and stones**
- **Scoring system**
  - Score & High Score saved in `localStorage`
- **Retro-style Web Audio sounds**
  - Jump, Score, Die

## 🧠 Bot AI

The game includes an optional **Autobot** that plays automatically:

- 🪨 **Stone hopping logic** – precise jump & cut jump  
- 🦅 **Pterodactyl logic** – crouch under low flyers, ignore high ones  
- 🌵 **Cactus logic** – jumps over all cactus types  
- ⚡ **Reaction distances** scale with game speed for more accuracy
